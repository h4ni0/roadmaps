# خريطة البرمجة والذكاء الاصطناعي - فقه البرمجة

خريطة طريق شاملة تغطي أساسيات البرمجة وصولاً إلى المواضيع المتقدمة، تشمل:

- محو الأمية التقنية وأساسيات البرمجة
- بيئة التطوير وإدارة الإصدارات
- قواعد البيانات وبناء التطبيقات
- معمارية التطبيقات والأمن السيبراني
- علوم الحاسوب والشبكات
- هندسة البرمجيات والبنية التحتية
- التخصص (ويب، جوال، سطح المكتب)

## الهيكلية

كل خريطة طريق تتكون من:

- **ملف JSON** يحتوي على هيكل الخريطة والمواضيع
- **مجلد `content/`** يحتوي على شرح كل موضوع بملف Markdown مستقل
