# JSON

JSON (اختصار لـ JavaScript Object Notation) هو صيغة خفيفة لتبادل البيانات بين الأنظمة. يُستخدم بشكل واسع في واجهات البرمجة (APIs) لإرسال واستقبال البيانات بين الخادم (Server) والعميل (Client). يتميز JSON بسهولة القراءة للبشر وسهولة التحليل للآلات، ويتكون من أزواج مفتاح-قيمة (key-value pairs) ومصفوفات (arrays). تقريباً كل لغات البرمجة الحديثة تدعم التعامل مع JSON بشكل مباشر، مما جعله المعيار الأساسي لتبادل البيانات على الويب.
