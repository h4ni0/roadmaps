# ما هو الموقع؟

الموقع الإلكتروني (Website) هو مجموعة من الصفحات المترابطة المخزنة على خادم (Server) ويمكن الوصول إليها عبر الإنترنت باستخدام متصفح ورابط (URL). تُبنى المواقع باستخدام ثلاث تقنيات أساسية: HTML لهيكلة المحتوى، CSS لتنسيق المظهر، وJavaScript لإضافة التفاعلية. هناك نوعان رئيسيان: مواقع ثابتة (Static) تعرض نفس المحتوى لكل زائر، ومواقع ديناميكية (Dynamic) يتغير محتواها حسب المستخدم مثل فيسبوك وتويتر. فهم بنية المواقع مهم لأنها من أكثر المشاريع البرمجية شيوعاً ونقطة انطلاق ممتازة لتعلم البرمجة.
