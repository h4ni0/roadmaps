# GitHub

GitHub هو أكبر منصة لاستضافة مشاريع Git والتعاون البرمجي في العالم، ويستخدمه ملايين المطورين والشركات. يوفر GitHub واجهة ويب سهلة لإدارة المستودعات (Repositories)، ومراجعة الشيفرة عبر طلبات السحب (Pull Requests)، وتتبع المشكلات (Issues)، وإدارة المشاريع. بالإضافة إلى ذلك، يوفر GitHub أدوات مثل GitHub Actions للتكامل المستمر (CI/CD) و GitHub Pages لاستضافة المواقع الثابتة. بناء ملف شخصي قوي على GitHub يُعد من أفضل الطرق لعرض مهاراتك البرمجية أمام أصحاب العمل.
