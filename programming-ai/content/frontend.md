# الواجهة الأمامية

الواجهة الأمامية (Frontend) هي الجزء المرئي من التطبيق الذي يتفاعل معه المستخدم مباشرة في المتصفح أو على الهاتف. تُبنى الواجهات الأمامية للويب باستخدام ثلاث تقنيات أساسية: HTML لبناء هيكل الصفحة، و CSS لتنسيقها وتصميمها، و JavaScript لإضافة التفاعلية والحركة. في التطوير الحديث، تُستخدم أُطر عمل مثل React و Vue و Angular لبناء واجهات معقدة وتفاعلية بكفاءة عالية. المطور الأمامي مسؤول عن تجربة المستخدم (UX) وضمان أن التطبيق يعمل بسلاسة على مختلف الأجهزة والمتصفحات.
