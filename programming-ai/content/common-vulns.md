# أهم الثغرات الأساسية

الثغرات الأمنية هي نقاط ضعف في التطبيقات يمكن للمهاجمين استغلالها. من أشهرها: حقن SQL (SQL Injection) وهي إدخال أوامر SQL خبيثة عبر حقول الإدخال للوصول لقاعدة البيانات، والبرمجة عبر المواقع (XSS - Cross-Site Scripting) وهي حقن أكواد JavaScript في صفحات الويب لسرقة بيانات المستخدمين، وتزوير الطلبات عبر المواقع (CSRF - Cross-Site Request Forgery) وهي خداع المستخدم لتنفيذ إجراءات غير مرغوبة. كذلك هناك ثغرات مثل كشف البيانات الحساسة وضعف إدارة الجلسات. منظمة OWASP تنشر قائمة بأخطر 10 ثغرات أمنية بشكل دوري وهي مرجع أساسي لكل مطور.
