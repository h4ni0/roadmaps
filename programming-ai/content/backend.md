# الواجهة الخلفية

الواجهة الخلفية (Backend) هي الجزء غير المرئي من التطبيق الذي يعمل على الخادم (Server) ويتولى معالجة البيانات وتنفيذ منطق العمل (Business Logic). يتعامل الـ Backend مع قواعد البيانات، ومعالجة الطلبات القادمة من الواجهة الأمامية، وإدارة المصادقة والصلاحيات، وإرسال الاستجابات المناسبة. من أشهر لغات وتقنيات الـ Backend: Node.js (JavaScript)، و Python (Django/Flask)، و Java (Spring)، و PHP (Laravel). يُعتبر فهم الواجهة الخلفية ضرورياً لبناء تطبيقات آمنة وقابلة للتوسع.
