# كتابة كود نظيف

الكود النظيف هو كود سهل القراءة والفهم والصيانة. يشمل ذلك اختيار أسماء واضحة ومعبّرة للمتغيرات والدوال، وكتابة دوال قصيرة تقوم بمهمة واحدة، وتجنّب التكرار (DRY)، وإضافة تعليقات فقط عند الضرورة. كتابة كود نظيف ليست رفاهية بل ضرورة، لأن المبرمج يقضي وقتاً في قراءة الكود أكثر بكثير مما يقضيه في كتابته.
