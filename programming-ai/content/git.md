# Git

Git هو نظام إدارة إصدارات موزّع (Distributed Version Control System) وهو الأكثر استخداماً في العالم. يتميز Git بأنه يحتفظ بنسخة كاملة من تاريخ المشروع على جهاز كل مطور، مما يجعل العمل سريعاً حتى بدون اتصال بالإنترنت. يتيح لك Git إنشاء فروع (Branches) للعمل على ميزات جديدة بشكل مستقل، ثم دمجها (Merge) مع الفرع الرئيسي بعد الانتهاء. تعلّم أوامر Git الأساسية مثل commit و push و pull و branch يُعتبر مهارة لا غنى عنها لأي مبرمج.
