# GitLab

GitLab هو منصة متكاملة لإدارة دورة حياة تطوير البرمجيات (DevOps)، تشبه GitHub في استضافة مستودعات Git لكنها تتميز بتوفير أدوات CI/CD مدمجة بشكل أصيل في المنصة. يمكن استخدام GitLab كخدمة سحابية أو تثبيته على خوادمك الخاصة (Self-hosted)، مما يجعله خياراً مفضلاً للشركات التي تحتاج تحكماً كاملاً في بياناتها. يدعم GitLab مراجعة الشيفرة، وتتبع المشكلات، وإدارة الحاويات (Container Registry)، والنشر التلقائي، كل ذلك في منصة واحدة.
