# مراجعة الكود - Code Review

مراجعة الكود هي عملية فحص الكود المكتوب من قبل مبرمج آخر قبل دمجه في المشروع الأساسي، وتُعد من أهم ممارسات هندسة البرمجيات لتحسين جودة الكود واكتشاف الأخطاء مبكراً. المراجعة الفعّالة لا تقتصر على البحث عن الأخطاء البرمجية، بل تشمل التأكد من وضوح الكود وقابليته للقراءة والصيانة، والتحقق من اتباع معايير المشروع، ومناقشة القرارات التصميمية. تعلّم كيف تعطي ملاحظات بنّاءة وكيف تتقبل الملاحظات على كودك مهارة أساسية للعمل ضمن فريق.
