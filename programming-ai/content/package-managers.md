# مدير الحزم والاعتماديات - Package Managers

مدير الحزم (Package Manager) هو أداة تُسهّل تثبيت وإدارة وتحديث المكتبات والاعتماديات (Dependencies) التي يحتاجها مشروعك البرمجي. من أشهر الأمثلة: npm و yarn لـ JavaScript، و pip لـ Python، و Composer لـ PHP، و Maven/Gradle لـ Java. يتضمن العمل مع مدير الحزم فهم ملفات التعريف (مثل package.json أو requirements.txt)، وملفات القفل (Lock Files) التي تضمن تثبيت نفس الإصدارات في كل مكان، وإصدارات الحزم (Semantic Versioning)، وكيفية تقييم المكتبات قبل استخدامها في مشروعك.
