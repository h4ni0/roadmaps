# أمان كود الـAI

مع تزايد استخدام أدوات الذكاء الاصطناعي مثل GitHub Copilot وChatGPT في كتابة الكود، ظهرت تحديات أمنية جديدة يجب الانتباه لها. الكود المُولَّد بالذكاء الاصطناعي قد يحتوي على ثغرات أمنية مثل حقن SQL أو استخدام مكتبات قديمة بها ثغرات معروفة، أو تسريب معلومات حساسة أو أنماط برمجية غير آمنة. يجب على المبرمج مراجعة كل سطر يولده الذكاء الاصطناعي بعناية، واستخدام أدوات تحليل الكود الثابت (Static Analysis) مثل Snyk وSonarQube، وعدم الثقة العمياء بأي كود مهما كان مصدره.
