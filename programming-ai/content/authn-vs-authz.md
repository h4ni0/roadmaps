# الفرق بين Authentication و Authorization

المصادقة (Authentication) والتفويض (Authorization) مفهومان مختلفان لكن مرتبطان ببعض. المصادقة (AuthN) هي عملية التحقق من هوية المستخدم - يعني "مين أنت؟" - وتتم عادةً من خلال اسم المستخدم وكلمة المرور أو بصمة الإصبع أو رمز مؤقت. أما التفويض (AuthZ) فهو عملية التحقق من الصلاحيات - يعني "إيش تقدر تسوي؟" - وتتم بعد المصادقة لتحديد الموارد والعمليات المسموح بها للمستخدم. مثلاً، لما تسجل دخول في موقع (Authentication)، النظام يحدد إذا كنت مدير أو مستخدم عادي (Authorization) ويعرض لك الصفحات المسموح لك فيها فقط.
