# قاعدة البيانات العلائقية

قاعدة البيانات العلائقية (Relational Database) تُنظّم البيانات في جداول (Tables) مترابطة عبر مفاتيح أساسية وأجنبية، وتستخدم لغة SQL للاستعلام والتعامل مع البيانات. من أشهر أنظمة قواعد البيانات العلائقية: PostgreSQL المعروف بقوته ودعمه للميزات المتقدمة، و MySQL الأكثر انتشاراً في تطبيقات الويب، و SQLite المناسب للتطبيقات الصغيرة والمحمولة. تتميز قواعد البيانات العلائقية بضمان تكامل البيانات (Data Integrity) من خلال القيود (Constraints) والمعاملات (Transactions) التي تتبع خصائص ACID.
