# هياكل البيانات الأساسية

هياكل البيانات الأساسية هي الطرق الأولى التي يتعلمها المبرمج لتنظيم وتخزين مجموعات من البيانات. تشمل المصفوفات (Arrays) لتخزين عناصر مرتبة، والقوائم (Lists) للتعامل مع بيانات متغيرة الحجم، والقواميس أو جداول التجزئة (Hash Maps / Dictionaries) لربط مفاتيح بقيم، والمجموعات (Sets) لتخزين عناصر فريدة. فهم هذه الهياكل ومتى تستخدم كل واحدة منها هو أساس كتابة أي برنامج حقيقي، من قائمة مهام بسيطة إلى نظام كامل.
