# Cloud - الحوسبة السحابية

الحوسبة السحابية هي نموذج يتيح الوصول إلى موارد الحوسبة (خوادم، تخزين، قواعد بيانات، شبكات) عبر الإنترنت حسب الطلب، بدلاً من امتلاك وإدارة خوادم فعلية. تقدم المنصات الرئيسية مثل AWS وGoogle Cloud وAzure خدمات متنوعة تشمل الاستضافة والحوسبة بدون خادم (Serverless) وقواعد البيانات المُدارة وخدمات الذكاء الاصطناعي. فهم نماذج الخدمة المختلفة (IaaS, PaaS, SaaS) ومبادئ تصميم الأنظمة السحابية مثل التوسع الأفقي والتوافر العالي يساعدك على اتخاذ قرارات معمارية صحيحة لتطبيقاتك.
