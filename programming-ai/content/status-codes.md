# رموز الحالة - Status Codes

رموز الحالة (Status Codes) هي أرقام يرجعها الخادم (Server) مع كل استجابة HTTP لتوضيح نتيجة الطلب. تنقسم إلى خمس فئات رئيسية: الفئة 1xx للمعلومات، و 2xx للنجاح (مثل 200 OK و 201 Created)، و 3xx لإعادة التوجيه (مثل 301 Moved Permanently)، و 4xx لأخطاء العميل (مثل 404 Not Found و 401 Unauthorized و 403 Forbidden)، و 5xx لأخطاء الخادم (مثل 500 Internal Server Error). معرفة هذه الرموز تساعدك في تشخيص المشاكل وبناء تطبيقات تتعامل مع الأخطاء بشكل صحيح.
