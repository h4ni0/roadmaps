# CI/CD

التكامل المستمر (CI) والنشر المستمر (CD) هما ممارستان أساسيتان في هندسة البرمجيات الحديثة تهدفان إلى أتمتة عملية بناء واختبار ونشر التطبيقات. التكامل المستمر يعني دمج التغييرات بشكل متكرر مع تشغيل الاختبارات تلقائياً للتأكد من عدم كسر أي شيء، بينما النشر المستمر يعني نشر التحديثات إلى بيئة الإنتاج بشكل آلي بعد اجتياز جميع الاختبارات. أدوات مثل GitHub Actions وGitLab CI وJenkins تُسهّل بناء خطوط أنابيب (pipelines) فعّالة تسرّع دورة التطوير وتقلل الأخطاء البشرية.
