# HTTP/HTTPS

بروتوكول HTTP (HyperText Transfer Protocol) هو البروتوكول الأساسي لنقل البيانات على الويب، ويعمل بنظام الطلب والاستجابة (Request/Response) بين المتصفح والخادم. يتضمن أنواعًا مختلفة من الطلبات مثل GET للقراءة وPOST للإرسال وPUT للتحديث وDELETE للحذف، إضافة إلى رموز الحالة (Status Codes) مثل 200 و404 و500. أما HTTPS فهو النسخة الآمنة التي تُشفّر البيانات باستخدام TLS لحمايتها من التنصت والتعديل أثناء النقل. فهم HTTP/HTTPS ضروري لكل مبرمج ويب بدون استثناء.
