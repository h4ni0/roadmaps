# كيف يُنفَّذ البرنامج

عندما تكتب كودًا وتقوم بتشغيله، يمر البرنامج بعدة مراحل قبل أن يُنفَّذ فعليًا على المعالج. في اللغات المُجمَّعة (Compiled) مثل C يتم تحويل الكود إلى لغة الآلة عبر المُجمِّع (Compiler)، بينما في اللغات المُفسَّرة (Interpreted) مثل Python يتم تنفيذ الكود سطرًا بسطر عبر المُفسِّر (Interpreter). يقوم نظام التشغيل بتحميل البرنامج إلى الذاكرة وإنشاء عملية (Process) لها مساحتها الخاصة من الذاكرة والموارد، ثم يبدأ المعالج بتنفيذ التعليمات واحدة تلو الأخرى.
