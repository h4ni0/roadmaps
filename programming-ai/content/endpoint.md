# نقطة الوصول - Endpoint

نقطة الوصول (Endpoint) هي عنوان URL محدد في واجهة برمجة التطبيقات (API) يمثل مورداً أو خدمة معينة يمكنك التواصل معها. مثلاً، لو عندك تطبيق لإدارة المستخدمين، ممكن يكون عندك نقطة وصول مثل `/api/users` لجلب قائمة المستخدمين، و `/api/users/1` لجلب بيانات مستخدم معين. كل نقطة وصول تستقبل طلبات HTTP بأنواعها المختلفة (GET, POST, PUT, DELETE) وترجع استجابة معينة. فهم نقاط الوصول ضروري لأي مطور يريد بناء أو استخدام APIs.
