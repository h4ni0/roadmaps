# REST API

واجهة برمجة التطبيقات REST (Representational State Transfer) هي أسلوب معماري لتصميم واجهات برمجية تسمح للتطبيقات بالتواصل مع بعضها عبر بروتوكول HTTP. يعتمد REST على مفهوم الموارد (Resources) التي يتم التعامل معها باستخدام أفعال HTTP المعروفة: GET للقراءة، POST للإنشاء، PUT/PATCH للتحديث، و DELETE للحذف. تتميز واجهات REST بأنها عديمة الحالة (Stateless)، أي أن كل طلب يحتوي على جميع المعلومات اللازمة لمعالجته. تُعد REST API المعيار الأكثر شيوعاً للتواصل بين الواجهة الأمامية والخلفية في تطبيقات الويب والهاتف.
