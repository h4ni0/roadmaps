# Docker

Docker هو منصة لتعبئة التطبيقات داخل حاويات (Containers) معزولة تحتوي على كل ما يحتاجه التطبيق للعمل، مما يضمن أن التطبيق يعمل بنفس الطريقة على أي جهاز أو خادم. باستخدام Docker، يمكنك تعريف بيئة التطبيق في ملف Dockerfile وإدارة خدمات متعددة باستخدام Docker Compose، مما يُسهّل عملية التطوير المحلي والنشر. فهم مفاهيم مثل الصور (Images) والحاويات (Containers) والشبكات (Networks) والمجلدات (Volumes) ضروري لأي مبرمج يريد العمل في بيئات حديثة.
