# سطر الأوامر

سطر الأوامر (Command Line / Terminal) هو واجهة نصية تتيح لك التفاعل مع نظام التشغيل وتنفيذ الأوامر مباشرة دون الحاجة إلى واجهة رسومية. يُعد إتقان سطر الأوامر مهارة أساسية لكل مبرمج، حيث يُستخدم في تشغيل البرامج وإدارة الملفات والمجلدات وتثبيت الحزم والمكتبات والتعامل مع Git. تختلف أنظمة سطر الأوامر حسب نظام التشغيل: Bash في لينكس وماك، وPowerShell أو CMD في ويندوز، لكن المفاهيم الأساسية مثل التنقل بين المجلدات ونسخ الملفات وتشغيل السكربتات تبقى متشابهة.
