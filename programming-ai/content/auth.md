# المصادقة

المصادقة (Authentication) هي عملية التحقق من هوية المستخدم للتأكد من أنه هو فعلاً من يدّعي أنه هو، وتختلف عن التفويض (Authorization) الذي يحدد ما يُسمح للمستخدم بفعله. تشمل طرق المصادقة الشائعة: تسجيل الدخول بالبريد الإلكتروني وكلمة المرور، والمصادقة عبر مزودين خارجيين مثل Google و GitHub (OAuth)، والمصادقة الثنائية (2FA). في التطبيقات الحديثة، تُستخدم آليات مثل JWT (JSON Web Tokens) وجلسات الخادم (Sessions) لإدارة حالة تسجيل الدخول. كما توجد خدمات جاهزة مثل Firebase Auth و Auth0 تُسهّل تطبيق المصادقة دون بنائها من الصفر.
