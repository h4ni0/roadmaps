# الواجهات البرمجية - API

الواجهة البرمجية (API - Application Programming Interface) هي مجموعة من القواعد والأدوات التي تسمح لبرنامجين مختلفين بالتواصل مع بعضهما البعض. فكر فيها كنادل في مطعم: أنت (العميل) تطلب من النادل (API) طبقاً معيناً، والنادل يوصل طلبك للمطبخ (الخادم) ثم يعيد لك الطبق (البيانات). في البرمجة، تستخدم الـ APIs يومياً: عندما يعرض تطبيق حالة الطقس فهو يستخدم API لخدمة طقس، وعندما تسجل دخول بحساب Google فهو يستخدم Google API. فهم الواجهات البرمجية أساسي لأن التطبيقات الحديثة نادراً ما تعمل بمعزل عن الخدمات الأخرى، والتواصل بينها يتم عبر APIs.
