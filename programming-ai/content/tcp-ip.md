# TCP/IP

بروتوكول TCP/IP هو مجموعة البروتوكولات الأساسية التي يقوم عليها الإنترنت بالكامل. يتكون من طبقات متعددة: طبقة التطبيق (Application)، وطبقة النقل (Transport) حيث يعمل بروتوكول TCP، وطبقة الإنترنت (Internet) حيث يعمل بروتوكول IP، وطبقة الوصول للشبكة (Network Access). يضمن TCP تسليم البيانات بشكل موثوق ومرتب عبر آلية المصافحة الثلاثية (Three-way Handshake) وإعادة الإرسال عند فقدان الحزم. فهم TCP/IP أساسي لأي مبرمج يتعامل مع الشبكات أو يبني تطبيقات ويب.
