# Scalability - التوسع

التوسّع (Scalability) هو قدرة النظام على التعامل مع زيادة الحمل سواء بزيادة عدد المستخدمين أو حجم البيانات. يوجد نوعان رئيسيان: التوسّع الرأسي (Vertical Scaling) بزيادة قدرات الخادم الواحد، والتوسّع الأفقي (Horizontal Scaling) بإضافة خوادم جديدة. التوسّع الأفقي هو الأكثر شيوعاً في الأنظمة الكبيرة لأنه لا يملك سقفاً محدوداً، ويتطلّب فهم تقنيات مثل Load Balancing وDatabase Sharding والتخزين المؤقت (Caching).
