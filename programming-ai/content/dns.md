# DNS

نظام أسماء النطاقات (DNS - Domain Name System) هو بمثابة دليل الهاتف للإنترنت، حيث يُحوّل أسماء المواقع التي نكتبها مثل google.com إلى عناوين IP الرقمية التي تفهمها الحواسيب. عند كتابة عنوان موقع في المتصفح، يمر الطلب عبر عدة خوادم DNS بدءًا من المحلل التكراري (Recursive Resolver) مرورًا بخوادم الجذر (Root Servers) وصولًا لخادم النطاق المسؤول. فهم DNS يُساعدك في تشخيص مشاكل الاتصال وإعداد النطاقات وفهم كيفية عمل الإنترنت من الداخل.
